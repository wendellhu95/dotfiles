# dotfiles

## Installation

Do this in your home directory.

1\. Clone the repository.

```shell
git clone https://github.com/wendellhu95/dotfiles.git .dotfiles
```

2\. Link files.

```shell
ln -fs .dotfiles/.vimrc .vimrc \
	&& ln -fs .dotfiles/.zshrc .zshrc
	&& ln -fs .tmux.conf/.zshrc .tmux.conf
```
